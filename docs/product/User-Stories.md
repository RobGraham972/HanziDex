# User Stories (MVP)

## Beginner (Maya)
- As a new user, I want to discover a few beginner items daily so I can build momentum.
- As a learner, I want a clear “Catch it” action to convert discoverables to my Dex.
- As a learner, I want simple training with reveal → grade so I’m not overwhelmed.
- As a learner, I want examples to understand usage.

## Intermediate (Leo)
- As a learner, I want to filter by HSK, kind, and tags to focus my study.
- As a learner, I want to bury siblings and suspend leeches to reduce interference.
- As a learner, I want a dashboard showing my retention and workload trends.

## Power User (Xiao)
- As a power user, I want keyboard shortcuts in training to move quickly.
- As a power user, I want search, filter, and sort to be fast on large lists.
- As a power user, I want to tweak desired retention and daily limits.

## Accessibility
- As a keyboard user, I need clear focus states and predictable tab order.
- As a motion-sensitive user, I need reduced-motion support.
